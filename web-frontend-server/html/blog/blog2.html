Docker và Container Technology

Ngày đăng: 17/11/2025 | Tác giả: Pham Gia Huy & Pham Khanh Linh

Docker là gì?

Docker là nền tảng mã nguồn mở giúp đóng gói ứng dụng và tất cả dependencies vào một container độc lập. Container nhẹ hơn, khởi động nhanh hơn so với máy ảo truyền thống.

Sự khác biệt: Container vs Virtual Machine

Tiêu chí Container Virtual Machine
Khởi động Vài giây Vài phút
Kích thước MB GB
Performance Gần native Chậm hơn
Cô lập Process-level Full isolation

Các thành phần chính của Docker

  • Image: Template chứa OS + code + dependencies
  • Container: Instance chạy từ image
  • Dockerfile: File script để build image
  • Docker Compose: Tool quản lý nhiều container

Ví dụ lệnh cơ bản

# Pull image từ Docker Hub
docker pull nginx:latest

# Chạy container
docker run -d -p 80:80 --name my-nginx nginx

# Xem container đang chạy
docker ps

# Stop container
docker stop my-nginx

Kết luận: Docker đã cách mạng hóa cách deploy ứng dụng, giúp DevOps workflow trở nên đơn giản và hiệu quả hơn.
